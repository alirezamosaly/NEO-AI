2. Datenschutz auf einen Blick

2.1 Allgemeine Hinweise

Die SmartCopy-Erweiterung verarbeitet alle Daten lokal auf Ihrem Gerät. Es werden keine Daten an externe Server übertragen.

2.2 Datenspeicherung

Die Erweiterung speichert folgende Daten lokal:

  • Ihre bevorzugte OCR-Sprache
  • Zuletzt kopierte Texte (nur in der Zwischenablage)

3. Datenerfassung

3.1 Lokale Verarbeitung

Alle Funktionen der Erweiterung werden lokal auf Ihrem Gerät ausgeführt:

  • Textkopieren: Direkt in die Zwischenablage
  • Bild-OCR: Lokale Verarbeitung mit Tesseract.js
  • Spracheinstellungen: Lokale Speicherung

3.2 Keine Serverkommunikation

Die Erweiterung kommuniziert nicht mit externen Servern. Alle Daten bleiben auf Ihrem Gerät.

4. Ihre Rechte

Gemäß DSGVO haben Sie folgende Rechte:

  • Recht auf Auskunft
  • Recht auf Berichtigung
  • Recht auf Löschung
  • Recht auf Einschränkung der Verarbeitung
  • Recht auf Datenübertragbarkeit
  • Widerspruchsrecht

5. Datensicherheit

Die Erweiterung implementiert folgende Sicherheitsmaßnahmen:

  • Lokale Datenverarbeitung
  • Minimale Berechtigungen
  • Keine Datenübertragung
  • Regelmäßige Sicherheitsupdates

6. Änderungen der Datenschutzerklärung

Wir behalten uns vor, diese Datenschutzerklärung anzupassen. Die aktuelle Version finden Sie stets in der Erweiterung.
